  • MY EXAMPLE: astroquery_messineo_v3.py
    This macro reads the name of a Vizier Catalog (assumed to be IR)
    find the 2MASS matches
    find the corresponding Gaia Dr3 source_id
    check and plot the BP-RP spectra
    outputs: eps figures, one per spectrum.
    IT IS a SYNCRON. JOB FOR SMALL TABLES. IN ORDER TO USE YOU MUST TEPLACE THE "user_mmessi01" user account name, AND YOU MUST HAVE An ESA/ARCHIVE USER ACCOUNT AND PASSWORD.
  • BPRP-Datalink: datalink_BPRP.py
    Attention: not all BPRP spectra are given in the SAMPLED.
    Given a list of Gaia DR3 source_IDs,
    it retrieves available SAMPLED BPRP spectra.
    The spectra (one per source_IDs) are saved individually in cvs file and
    also an alltogheter.DTL.csv file (unique) is generated.
    Note that the macro is made by two parts: comment one or the other if you want the alltoghether or the
    individual file.
    Also note that for the bulk retrieval you need an account and password.
    For the individual retrieval it is not needed (free access).
  • BPRP-make your own sampling with GaiaXpy: defaultsampling_BPRP.py
    Given a list of Gaia DR3 source_IDs,
    it retrieves available continous BPRP spectra and resample them.
